# Профиль личности и успеха

Символический профайлер на основе выбора продуктов: носки, бритва, виды спорта. Определяет архетип и даёт структурированный профиль (не является финансовой консультацией).

## Требования

- Node.js 18+
- npm

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173

## Сборка

```bash
npm run build
npm run preview
```

## Язык

Весь интерфейс на русском языке. Стек: React, TypeScript, Vite, React Router. Состояние — в контексте (сессия), без сохранения персональных данных.
